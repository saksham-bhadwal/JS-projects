# JavaScript Projects Portfolio

This repository contains some beginner JavaScript projects I built while learning frontend development.
These projects helped me practice **DOM manipulation, APIs, localStorage, and game logic** using **HTML, CSS, and JavaScript**.

---

## 🚀 Projects

### 1️⃣ Todo App

A simple task manager that allows users to add and manage daily tasks.

**Features**

* Add tasks
* Delete tasks
* Tasks saved using **localStorage**
* Data persists after page refresh

**Tech Used**

* HTML
* CSS
* JavaScript
* localStorage

---

### 2️⃣ Notes App

A minimal notes application where users can quickly write and save notes.

**Features**

* Create notes
* Save notes automatically
* Notes stored in **localStorage**
* Clean UI

**Tech Used**

* HTML
* CSS
* JavaScript
* localStorage

---

### 3️⃣ Weather App

A weather application that shows current weather data for a searched city.

**Features**

* Search weather by city
* Displays temperature and weather conditions
* Fetches data from a weather API

**Tech Used**

* HTML
* CSS
* JavaScript
* Weather API

---

### 4️⃣ Movie Search App

An app that allows users to search for movies and view movie details.

**Features**

* Search movies by title
* Displays movie poster, rating, and description
* Uses movie API for data

**Tech Used**

* HTML
* CSS
* JavaScript
* Movie API

### 5️⃣ Rock Paper Scissors Game

A simple browser game where the user plays against the computer.
